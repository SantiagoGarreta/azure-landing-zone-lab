# Networking Module

## Purpose

This module will define the Azure networking foundation for the lab.

## Planned Resources

- Virtual Network
- Subnets
- Network Security Groups
- Optional subnet associations

## Inputs And Outputs

Inputs and outputs will be implemented later as the Terraform module is created.

## Security Considerations

- Use subnet-level segmentation to limit blast radius
- Restrict traffic with explicit NSG rules
- Avoid overly permissive ingress and egress definitions
